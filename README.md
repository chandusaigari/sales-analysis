# 🧥 Men's Clothing Sales Analysis - [dowload](https://github.com/chandusaigari/sales-analysis/blob/main/Insights.docx)

## 📌 Overview
Analyzed 6,000 records from January to March to generate insights on sales performance, customer satisfaction, and operations.

- Python → Data Cleaning & Preprocessing  
- Power BI → Data Visualization & Dashboard  

---

## 🎯 Key Insights
- 🛍️ Top product categories drive the highest revenue  
- 🌍 High-performing cities show better sales and ratings  
- 🚚 Faster delivery improves customer satisfaction  
- 💳 Payment methods influence order value  
- 👕 Size and color preferences impact sales
